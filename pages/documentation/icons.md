# Icons

React
-----------

Template makes use of heroicons icon set. See official docs here <https://heroicons.com/>

> Import solid icon
```js
import {
  ChevronLeft
} from "icons/solid";
```

> Import outline icon
```js
import {
  ChevronLeft
} from "icons/outline";
```

> Usage
```html
<ChevronLeft width={16} height={16} strokeWidth={2} />
```
